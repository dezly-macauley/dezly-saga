## How to create a new project

```
mkdir name-of-project
cd name-of-project
mox init --pyproject
```

This will create a new Moccasin project with a `pyproject.toml` file.

This is file is used by the package manager `uv`
_______________________________________________________________________________
## How to compile your project

This will compile every Vyper smart contract in the `src` directory

```
mox compile
```
_______________________________________________________________________________
## How to run tests

```
mox test
```
_______________________________________________________________________________
## How to deploy your contract

```
mox run deploy
```
